# todo

A simple todo app with Google login

## Development

```bash
npm install && npm run dev
```

## Deployment

Every push to `main` auto-deploys to **https://todo.asikmydeen.com** via Dokploy.

## Infrastructure

- **Hosting:** Dokploy on TrueNAS → https://projects.asikmydeen.com
- **Database:** Supabase → https://supabase.asikmydeen.com
- **Domain:** https://todo.asikmydeen.com
